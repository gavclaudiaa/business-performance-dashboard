# 2024 Business Performance Dashboard

## Project Overview
This project presents an interactive Business Performance Dashboard built using Power BI to analyze more than 10,000 synthetic retail transaction records. The dashboard evaluates overall business performance by tracking revenue, cost, profit, profit margin, customer segmentation, regional performance, discount impact, and payment method distribution. The objective of this project is to transform raw transaction data into actionable business insights through structured data modeling and KPI visualization.

## Dataset Description
The dataset contains 10,000+ structured retail transaction records. Each row represents a single sales transaction.
Columns included:
- Order_Date  
- Region  
- Product_Category  
- Customer_Segment  
- Quantity  
- Unit_Price  
- Discount_Rate  
- Revenue  
- Cost  
- Profit  
- Payment_Method  
All financial metrics are internally consistent. The dataset is synthetic and created for educational and analytical purposes.

## Business Questions Addressed
- What is the total revenue, cost, and profit performance?
- Which customer segment generates the highest profitability?
- How do discount rates impact profit?
- How does business performance fluctuate monthly?
- Which payment method contributes the highest revenue?

## Key Features of the Dashboard
- KPI cards for Total Revenue, Total Cost, Total Profit, and Profit Margin  
- Revenue achievement gauge visualization  
- Profitability breakdown by customer segment  
- Monthly revenue, cost, and profit trend analysis  
- Scatter analysis of discount rate versus profit  
- Revenue distribution by payment method  
- Interactive filters for Region, Product Category, Customer Segment, and Payment Method  

## Tools and Technologies
- Power BI  
- Microsoft Excel  
- Data Modeling (Star Schema concept)  
- DAX measures for KPI calculations  

## Key Insights
- Revenue and profit contributions are relatively balanced across customer segments.
- Higher discount rates are associated with lower profit margins.
- Payment methods show varied revenue contributions, indicating differences in customer purchasing behavior.
- Monthly performance trends indicate relatively stable revenue with seasonal fluctuations.
